🔒 隐私政策 - SmartRead

最后更新：2026年5月25日

数据收集

SmartRead 不收集、不存储、不传输任何个人数据。所有 AI 分析功能完全在你的浏览器中运行。

我们访问的内容

  • 网页选中文本：仅当你在网页上选中文本并触发分析时，选中的内容会临时发送至 DeepSeek API 进行处理，处理完成后立即丢弃，不会在本地或服务器上存储。
  • API Key：你的 DeepSeek API Key 仅存储在你的浏览器本地存储（chrome.storage.local）中，不会上传到任何服务器。

权限说明

  • storage：用于在本地保存你的 API Key 和偏好设置。
  • activeTab：仅在点击扩展图标时访问当前网页内容。
  • contextMenus：用于在右键菜单中显示 SmartRead 快捷操作。

第三方服务

SmartRead 使用 DeepSeek API（api.deepseek.com）进行文本分析。发送到 DeepSeek 的数据仅为你选中要求分析的网页文本，DeepSeek 的隐私政策请参考 platform.deepseek.com/privacy。

信息安全

所有扩展程序数据仅存储在你的本地浏览器中，不通过任何外部服务器中转。我们不会出售、分享或以任何形式泄露你的数据。
